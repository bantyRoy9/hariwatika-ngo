## рҹҺ¬ User Flow Examples

### Adding a Timeline Event

1. Admin logs in вҶ’ Goes to `/admin/page-editor/about`
2. Sees yellow **"Add Timeline Event"** button
3. Clicks button вҶ’ Form modal opens
4. Fills in:
   - Year: `2024`
   - Event (EN): `Launched new health initiative`
   - Event (HI): `аӨЁаӨҲ аӨёаҘҚаӨөаӨҫаӨёаҘҚаӨҘаҘҚаӨҜ аӨӘаӨ№аӨІ аӨ¶аҘҒаӨ°аҘӮ аӨ•аҘҖ`
5. Clicks **"Add Event"**
6. Page reloads вҶ’ New event appears in timeline

### Editing a Team Member

1. Admin hovers over member card
2. Sees вңҸпёҸ (yellow pencil) and рҹ—‘пёҸ (red trash) icons appear
3. Clicks вңҸпёҸ edit icon вҶ’ Form modal opens with current data
4. Updates:
   - Name: `Rajesh Kumar Singh` (was `Rajesh Kumar`)
   - Designation: `Vice President` (was `Director`)
5. Clicks **"Update Member"**
6. Page reloads вҶ’ Member card shows new info + new initials (RKS)

### Deleting a Legal Document

1. Admin hovers over document card
2. Sees вңҸпёҸ and рҹ—‘пёҸ icons appear
3. Clicks рҹ—‘пёҸ delete icon
4. Browser shows confirmation: **"Are you sure you want to delete this item?"**
5. Admin clicks **OK**
6. Page reloads вҶ’ Document card is gone

## рҹ”җ Security Behavior

### When Admin is Logged In

**On `/about` (public page):**
- вқҢ NO add buttons
- вқҢ NO edit/delete icons
- вң… Content displays normally

**On `/admin/page-editor/about?editMode=true`:**
- вң… Add buttons appear
- вң… Edit/delete icons appear on hover
- вң… All forms work
- вң… CRUD operations allowed

### When User is NOT Logged In

**On any page:**
- вқҢ NO add buttons
- вқҢ NO edit/delete icons
- вқҢ Server actions return 401 Unauthorized
- вң… Content displays normally
