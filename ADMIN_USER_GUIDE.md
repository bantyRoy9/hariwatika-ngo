### ✏️ Page Editor

**Purpose:** Edit text content on any page without touching code

**How to use:**

1. **Click "Page Editor"** in sidebar
2. **Select a page** from dropdown (Home, About, Blog, etc.)
3. **Page loads in preview** - you'll see the actual website
4. **Double-click any text** to edit it
5. **Edit in the popup:**
   - English text (top field)
   - Hindi text (bottom field)
6. **Click "Save"** - changes save immediately
7. **Click "Refresh Preview"** to see updated content

**Pages you can edit:**
- Home Page
- About Us
- Projects
- Blog
- Transparency
- Contact
- Volunteer
- Donate
- Gallery
- Programs
- Internship

**What you CANNOT edit here:**
- Project cards (use Content > Projects)
- Blog posts (use Content > Blog)
- Team members (use Content > About)
- Images (use Media Library)

### 🖼️ Media Library
**Path:** Media

**Purpose:** Upload and manage all images used on website

**How to upload images:**
1. Click "Upload Media"
2. Select images from computer (or drag & drop)
3. Add titles/descriptions
4. Click "Upload"

**Supported formats:**
- JPEG/JPG
- PNG
- WebP
- SVG (for icons)

**Image guidelines:**
- Maximum size: 5MB per image
- Recommended dimensions:
  - Hero images: 1920x1080px
  - Project photos: 800x600px
  - Team photos: 400x400px (square)
  - Blog featured images: 1200x630px

**How to use uploaded images:**
1. Click on image to copy URL
2. Paste URL in content editor
3. Image appears on website

## 🎨 Common Tasks

### Task 1: Update Homepage Hero Text

1. Go to **Page Editor**
2. Select **Home** from dropdown
3. Double-click the main heading
4. Edit English and Hindi text
5. Click **Save**
6. Click **Refresh Preview**
7. Verify changes

### Task 2: Add a New Project

1. Go to **Content > Projects**
2. Click **Add New Project**
3. Fill in all details:
   - Title (both languages)
   - Description (both languages)
   - Category
   - Location
   - Goal amount
4. Upload featured image
5. Click **Save**
6. Project appears on Projects page

### Task 3: Respond to Contact Form

1. Go to **Submissions > Contacts**
2. Click on submission to view
3. Read message
4. Click **Reply via WhatsApp** or **Reply via Email**
5. Send response
6. Mark as **Resolved**

### Task 4: Approve Volunteer

1. Go to **Submissions > Volunteers**
2. Review application
3. Click **Approve**
4. System generates Volunteer ID
5. Click **Send ID Card**
6. Volunteer receives ID via email/WhatsApp

### Task 5: Upload Event Photos

1. Go to **Media Library**
2. Click **Upload Media**
3. Select multiple photos
4. Add event name in title
5. Click **Upload**
6. Photos available for use

### Task 6: Post a Blog Update

1. Go to **Content > Blog**
2. Click **Add New Post**
3. Write title (EN/HI)
4. Write content using editor
5. Add category and tags
6. Upload featured image
7. Click **Publish**

### Task 7: Update Financial Data

1. Go to **Content > Transparency**
2. Click **Add Report** for new year
3. Fill in:
   - Year
   - Income
   - Expense
   - Beneficiaries
4. Upload annual report PDF
5. Click **Save**
6. Data appears on Transparency page

### Task 8: Send Festival Greetings

1. Go to **Submissions > Donations**
2. Select donors to send greetings
3. Click **Bulk Actions**
4. Select **Send Greetings**
5. Choose greeting template
6. Click **Send**
7. System sends via WhatsApp

## 🔒 Security Best Practices

### Password Security
- ✅ Use unique, strong password
- ✅ Change password every 90 days
- ✅ Never share password
- ✅ Don't write password down
- ✅ Log out when leaving computer

### Session Security
- ✅ Always log out after use
- ✅ Don't use on public computers
- ✅ Clear browser cache regularly
- ✅ Use HTTPS only (lock icon in URL)

### Data Protection
- ✅ Don't delete submissions without backup
- ✅ Export important data monthly
- ✅ Review activity logs weekly
- ✅ Report suspicious activity immediately

## 🐛 Troubleshooting

### Can't Login

**Problem:** "Invalid credentials" error

**Solutions:**
1. Check username spelling (case-sensitive)
2. Verify Caps Lock is OFF
3. Try "Forgot Password" link
4. Clear browser cache
5. Try different browser
6. Contact technical support

### Edit Mode Not Working

**Problem:** Can't see pencil icons in Page Editor

**Solutions:**
1. Verify you're logged in
2. Check you're on correct page (`/admin/page-editor/...`)
3. Refresh the iframe
4. Clear browser cache
5. Try logging out and back in

### Image Upload Failed

**Problem:** "Upload failed" error

**Solutions:**
1. Check file size (max 5MB)
2. Verify file format (JPG/PNG/WebP)
3. Check internet connection
4. Try different browser
5. Contact technical support

### Changes Not Appearing

**Problem:** Edited content doesn't show on live site

**Solutions:**
1. Click "Refresh Preview" in Page Editor
2. Clear browser cache (Ctrl+Shift+R)
3. Wait 1-2 minutes for cache to clear
4. Check you clicked "Save" button
5. Verify you're viewing correct page

### Forgot Password

**Problem:** Can't remember password

**Solutions:**
1. Click "Forgot Password" on login page
2. Enter your email address
3. Check email for reset link
4. Click link and set new password
5. If no email, contact technical support
